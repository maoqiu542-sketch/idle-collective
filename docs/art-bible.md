# Idle Collective — Art Bible

> 版本 2.0 | 《火山的女儿》暖色调手绘风格 | 适用于所有 UI / 美术资源

## 1. 视觉定位

| 关键词 | 说明 |
|--------|------|
| **温暖治愈** | 棕黄橙大地色系，羊皮纸质感，有温度的手绘感 |
| **故事书插画** | 柔和圆润的线条，非现代扁平，带一点童话感 |
| **有机自然** | 避免锐利边角，元素之间有呼吸感 |
| **轻松挂机** | 无压迫感，无红色警告频闪，视觉节奏平静 |

**参考风格**：《火山的女儿》+ 《星露谷物语》+ 《动物森友会》UI


## 8. 动效规范

| 动效 | 时长 | 缓动 |
|------|------|------|
| 按钮 hover 背景 | `150ms` | `ease` |
| 面板出现 | `200ms` | `ease-out` |
| 进度条填充 | `300ms` | `linear` |
| 浮动文字上升消失 | `1000ms` | `ease-out` |
| BOSS 血条减少 | `400ms` | `ease-in-out` |

**禁止**：超过 `500ms` 的阻塞性动画；避免 `bounce/elastic` 缓动

## 11. 禁止清单（勿引入）

- 外部图片资源（暂无美术，全 emoji）
- CSS 框架（Bootstrap / Tailwind）—— 保持手写 CSS
- 外部图标库（FontAwesome 等）
- 冷色调主题（深蓝、黑色背景）
- 非 4 倍数间距（如 7px、15px）
- 纯黑/纯白背景
- 锐利直角边框
- 高饱和霓虹色
