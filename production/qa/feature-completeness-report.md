# 功能完成度报告

日期：2026-03-28

## 总评

当前项目已经具备 V1 主循环可玩形态，但还没有达到“全部完成”的状态。

- 总体完成度：`约 72%`
- 已完成：`约 72%`
- 半完成：`约 18%`
- 未完成或待打磨：`约 10%`

## 已完成

- 全局策略预设与角色手动优先级互锁
- 聚落宜居度、发展度基础公式
- 招募站 + 商队刷新链路
- 科技基础链路
- Boss 定时刷新、挑战、核心零件回流
- 存档、读档、版本校验
- 建筑放置、建造进度、生产建筑产出
- 建筑升级预览、升级消耗、地图真实占地表现
- 招募站、研究台升级收益基础接线
- Windows 桌面版构建与打包

## 半完成

- 房屋、厨房、仓库、兵营的功能已经接线，但反馈还可以更直观
- 招募站、研究台虽然已有升级收益，但还缺更强的持续反馈
- UI 仍有局部旧文案和表现需要统一
- 需要一次完整的 30 到 45 分钟人工试玩验收

## 未完成或明确后置

- 地形深度编辑
- 装饰系统
- 更复杂的社交/隐私/幸福系统
- 大量 Boss、职业、科技扩展内容
- 正式发布级数值平衡

## 建筑专项修正

建筑系统此前是主要短板，本轮修复后：

- 修复前评估：`40% - 50%`
- 当前评估：`70% - 75%`

详情见 [building-system-issue-list.md](/D:/GameDesign/Projects/idle-collective/production/qa/building-system-issue-list.md)。

1. 做建筑与角色分工的可视化。
2. 做一次完整人工试玩并沉淀问题单。
