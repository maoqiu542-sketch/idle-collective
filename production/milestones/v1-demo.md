# V1 Demo 里程碑

## 目标

完成一个可以对外试玩的 V1 Demo，证明以下体验链路成立：

1. 玩家设定全局策略或角色优先级。
2. 聚落角色自动执行工作。
3. 资源、招募、科技和建造形成推进。
4. Boss 定时刷新并掉落核心零件。
5. 核心零件推动升级并进入下一轮规划。

## 成功标准

- 可以在 Windows 桌面环境稳定启动。
- 至少能完整体验一轮“规划 -> 执行 -> Boss -> 升级”的闭环。
- 关键状态都能清晰展示给试玩者。
- 不依赖开发者讲解，也能理解当前主循环。

## 范围

### 包含

- `docs/v1` 同步完成
- `production` 资产完成
- 主循环可玩
- 基本存档和读档
- 基本 QA 冒烟

### 不包含

- 深度地形编辑
- 装饰系统
- 大量 Boss、职业和科技扩容
- 正式发布级数值平衡

## 退出条件

- Demo 可以成功打包
- QA 清单有明确结果
- 已知问题可追踪
- 关键玩法无需额外说明即可理解
